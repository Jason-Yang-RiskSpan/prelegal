# prelegal

A dataset of legal document templates sourced from [Common Paper](https://github.com/commonpaper), intended for use as a base for system-generated legal documents.

## Contents

- `catalog.json` — index of all templates with name, description, and file path
- `template/` — legal document templates organized by agreement type

## Templates

| Directory | Agreement |
|---|---|
| `template/Mutual-NDA/` | Mutual Non-Disclosure Agreement |
| `template/CSA/` | Cloud Service Agreement |
| `template/SLA/` | Service Level Agreement |
| `template/DPA/` | Data Processing Agreement |
| `template/Design-Partner-Agreement/` | Design Partner Agreement |
| `template/PSA/` | Professional Services Agreement |
| `template/Partnership-Agreement/` | Partnership Agreement |
| `template/BAA/` | Business Associate Agreement |
| `template/Software-License-Agreement/` | Software License Agreement |
| `template/Pilot-Agreement/` | Pilot Agreement |
| `template/AI-Addendum/` | AI Addendum |
